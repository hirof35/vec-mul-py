Vector Scalar Multiplication SimulatorPygame を用いて、2次元ベクトルの実数倍（スカラー倍）を直感的に視覚化・操作できるシミュレーションツールです。

概要このアプリケーションは、基底となる2次元ベクトル（緑）に対して、画面下部のスライダで実数（スカラー値 $k$）を動的に変化させることで、合成されたベクトル（青）がどのように伸縮・反転するかをリアルタイムに観察できます。
数学的なベクトルの概念（特に線形代数の基礎）を視覚的に理解するための教育・実験用ツールとして設計されています。
✨ 機能特徴直感的なベクトル操作: グリッド上をクリックするだけで、基底ベクトルの先端位置（整数座標）を自在に変更可能。
滑らかなスライダ駆動: マウスドラッグによるリアルタイムなスカラー値（$-3.0 \sim +3.0$）の変更に対応。
正確な座標同期: ピクセル空間と数学的座標系のズレを排し、直感的なクリック位置へ正確に追従。
インジケーター搭載: スライダの中央（$k = 0$）にガイド線を配置し、ベクトルの反転（正負の境界）を明確化。
モダンなダークテーマ: 長時間の観察でも目が疲れにくいカラーパレットを採用。動作環境Python: 3.8 以上Pygame: 2.0.0 以上
🚀 使い方1. 依存ライブラリのインストール事前に Pygame がインストールされていない場合は、以下のコマンドを実行してください。Bashpip install pygame
2. アプリケーションの起動リポジトリのルートディレクトリで、スクリプトを実行します。Bashpython vector_mul.py
3. 操作方法ベクトルの変更: 上部のグリッドエリア（黒い領域）の任意の交点を左クリックすると、原点からその点へ向かう新しい基底ベクトル（緑）が設定されます。
スカラー値の変更: 下部のスライダのノブ（白）をドラッグ、またはレール上をクリックすることで、倍率を変更できます。緑の線: 元のベクトル $\vec{v}$青の線: スカラー倍されたベクトル $k\vec{v}$コードの構造vector_mul.py
├── main()          # メインループ、イベントハンドリング、座標演算、描画管理
└── Slider          # 完全にカプセル化された、再利用可能なスライダUIコンポーネント
グラフィックス設計: マジックナンバーを排除し、画面サイズやグリッド幅（PIXELS_PER_UNIT）を定数管理。解像度や内部仕様の変更に強い構造にリファクタリングされています。
状態管理: 描画処理がグローバル変数に依存しない疎結合設計（Slider.draw(surface)）を採用。
